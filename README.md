# Itacate Zacatecas — menú para GitHub Pages

Archivos listos para subir:

- `index.html`
- carpeta `assets/` con todas las imágenes del menú

Para publicarlo gratis:

1. Crea un repositorio en GitHub.
2. Sube `index.html` y la carpeta `assets`.
3. Entra a Settings → Pages.
4. En Branch selecciona `main` y carpeta `/root`.
5. Guarda y usa la liga que te dé GitHub Pages para generar tu QR.

No cambies los nombres de la carpeta `assets` ni de las imágenes, porque el HTML ya las llama con esas rutas.
